What changed in revision 5

The previous version made the learning module more independent. This version makes it more systematic, searchable, and ready for long-term use.

🔎

Searchable theorem library

Users can search by theorem name and filter by completion state, making the module feel more like a real resource library.

🧠

Review queue

Wrong answers are not lost. They feed into a review panel so learners know what still needs attention.

📝

Learning log

Each meaningful action writes to a recent activity log, helping users see what they just completed.

📦

Progress export

Progress can now be exported as structured data, which makes the system feel more complete and portfolio-ready.

New study modules

This version is organized like a study workspace rather than only a content page. Each part now has a clearer role.

Module 1

Theorem library

Browse, search, and filter theorem cards. This turns theorem study into an actual library experience.

  • Keyword search
  • All / learned / review filters
  • Click any theorem to load it
Module 2

Interactive lab

The active theorem stays at the center of the workspace, with geometry drawing, controls, and a short check.

  • Canvas demonstration
  • Parameter controls
  • Immediate theorem check
Module 3

Review + export panel

The system now remembers what needs revision and allows users to export their current study data.

  • Recent actions log
  • Review queue from mistakes
  • Progress export button

Learning workspace

Search a theorem on the left, study it in the center, and manage progress and review from the right panel.

Angle at the centre
