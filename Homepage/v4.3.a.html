New systems added in this version

This revision shifts the platform from “tracking and motivating” into “guiding and reflecting”.

System 1

Recommendation engine

The platform now prioritises the next theorem based on review needs, incomplete items, and favorites.

  • Prioritise review items first
  • Then unfinished theorems
  • Then favorites for reinforcement
System 2

Reflection tools

Notes and reports move the platform beyond answering questions and into recording understanding.

  • Notes saved per theorem
  • Reusable study reflections
  • Generated report summary
System 3

Mastery visibility

A theorem is no longer just learned or not learned. It now has an estimated strength level.

  • Mastery board for all theorems
  • Weak areas are easier to spot
  • Supports more realistic guidance
System 4

Plan-led study

A checklist now frames the session so the learner can see whether this study block achieved its intended goals.

  • Study checklist
  • Task completion visibility
  • Clearer session narrative
