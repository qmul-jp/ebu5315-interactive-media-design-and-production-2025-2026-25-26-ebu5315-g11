Platform Overview

Three study modes, one tracked journey

This version separates learning into Learn, Review, and Exam modes, while also tracking favorites, goals, session length, and achievements.

Learn mode

Focus on one theorem, manipulate the figure, and complete a guided check.

Review mode

Return directly to weak areas captured from wrong answers.

Exam mode

Try a mixed short test across all theorems to simulate cumulative learning.

Achievement layer

Unlock badges for completion, consistency, favorites, and clearing review tasks.

What changed in revision 6

The previous version made the workspace searchable and reviewable. This version turns it into a more complete, multi-mode study platform.

⭐

Favorites system

Theorem cards can now be starred, so users can build a personal shortlist of key topics.

🎯

Goal tracking

The workspace tracks meaningful study actions against a small session goal.

🧪

Mixed exam practice

A new exam mode draws mixed questions from across the theorem set, not just the active theorem.

🏅

Achievement badges

Progress now unlocks badges, which makes the system feel more like a complete learning product.

New systems added in this version

Revision 6 does not just add features. It restructures the workspace around learning mode, review flow, and progress motivation.

System 1

Mode-based learning

Users can now switch between Learn, Review, and Exam modes depending on what kind of study they need.

  • Current theorem learning
  • Targeted review queue
  • Mixed theorem test
System 2

Personal study tools

Favorites and action goals make the workspace feel tailored to the user rather than fixed for everyone.

  • Favorite theorem list
  • Session goal bar
  • More visible study ownership
System 3

Motivation layer

Achievements and session tracking reward consistent progress and make long-term use more meaningful.

  • Badge unlocking
  • Session timer
  • Progress visibility
System 4

Data continuity

All the core state still persists locally, so study progress behaves more like a persistent platform.

  • Saved themes and language
  • Saved library state
  • Exportable progress data

Study workspace

Use the left panel to choose content, the center to study in the right mode, and the right panel to manage progress, badges, and output.
